Privacy Policy – Easy Currency (EC)
Last Updated: July 12, 2026
1. Information We Collect
Easy Currency (EC) is a browser extension designed solely for real-time currency conversion. We only process the following limited data:
Your manually selected currency types and exchange rate preferences: These settings are stored locally in your browser via chrome.storage.local and will never be uploaded to any remote server.
Exchange rate query requests: We only send currency code parameters to public open exchange rate APIs. No browser identifiers, browsing history, personal identity or sensitive data will be attached to these requests.
This extension does NOT collect, read or store any sensitive personal data including browsing records, account passwords, bank card information, physical addresses or unique device identifiers.
2. Data Storage
All user preferences are saved locally on your browser only. We maintain no cloud database and perform no remote synchronization of user data.
3. Third-Party Services
This extension relies on public free exchange rate APIs to provide conversion functions. Such third-party interfaces only receive currency parameters without any user private information. You may review the independent privacy policy of each API provider separately.
4. Data Sharing & Disclosure
We will never sell, trade, rent or distribute any user-related data to advertisers, third-party corporations or external organizations for commercial or other purposes.
5. Your User Rights
You may clear all locally stored data of this extension at any time through your browser extension management page.
If you have any inquiries regarding this privacy policy or data processing, please contact us via email: xxx@shturl.
6. Compliance Statement
This extension fully complies with the Chrome Web Store Developer Program Policies. All data operations are limited to supporting the core currency conversion function, with no unauthorized extra data collection or tracking behavior.
